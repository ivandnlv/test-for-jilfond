# Тестовое задание для компании "Жилфонд"

Небольшое веб-приложение, которое взаимодействует с jsonplaceholder api

Стэк:
- vue 3 (script setup);
- typescript;
- scss (scoped styles);
- Vuex;
- Axios.

## Запуск
```
npm install
```
```
npm run dev
```
